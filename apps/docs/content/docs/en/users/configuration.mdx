## Core variables

| Variable | Default | When to change it |
|---|---|---|
| `OUTLINE_API_KEY` | required | Always set |
| `OUTLINE_BASE_URL` | `https://app.getoutline.com` | Use self-hosted endpoint |
| `OUTLINE_ALLOWED_ACTIONS` | `read,write,delete` | Restrict per agent role |
| `OUTLINE_REQUEST_TIMEOUT_MS` | `15000` | Slow networks / strict timeout policy |
| `OUTLINE_RETRY_COUNT` | `2` | High-latency or unstable upstream |
| `OUTLINE_ENABLE_CAPABILITY_PROBE` | `true` | Disable only when probing is undesirable |

## Lease settings

| Variable | Default | Notes |
|---|---|---|
| `OUTLINE_LEASE_STRATEGY` | `auto` | `auto`, `memory`, `data_attribute` |
| `OUTLINE_LEASE_ATTRIBUTE_ID` | none | Required for `data_attribute` |
| `OUTLINE_LEASE_DEFAULT_TTL_SECONDS` | `600` | Default lease duration |
| `OUTLINE_LEASE_MAX_TTL_SECONDS` | `7200` | Hard ceiling for lease extension |

## Practical profiles

### Read-only assistant

```bash
OUTLINE_ALLOWED_ACTIONS=read
OUTLINE_ENABLE_CAPABILITY_PROBE=true
```

### Team writer

```bash
OUTLINE_ALLOWED_ACTIONS=read,write
```

### Full operator

```bash
OUTLINE_ALLOWED_ACTIONS=read,write,delete
```

### Multi-process lease control

```bash
OUTLINE_LEASE_STRATEGY=data_attribute
OUTLINE_LEASE_ATTRIBUTE_ID=your_data_attribute_id
```
