## Option A: npm (recommended)

Install globally:

```bash
npm i -g @sigeemin/outline-mcp
outline-mcp
```

Or run directly without global install:

```bash
npx -y @sigeemin/outline-mcp
```

## Option B: run from source (monorepo)

```bash
git clone <repo-url>
cd outline-mcp
pnpm install
pnpm --filter @sigeemin/outline-mcp build
pnpm --filter @sigeemin/outline-mcp dev
```

## Required environment variables

```bash
OUTLINE_API_KEY=your_api_key
OUTLINE_BASE_URL=https://app.getoutline.com
OUTLINE_ALLOWED_ACTIONS=read,write,delete
```

## Quick verification

If you run from source, use:

```bash
pnpm --filter @sigeemin/outline-mcp build
pnpm e2e:mcp
```

This verifies tool wiring and runtime behavior.
