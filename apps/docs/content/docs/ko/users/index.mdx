## 첫 주 롤아웃 권장안

현업에서 안정적으로 시작하려면 보통 아래 순서가 가장 안전합니다.

1. 클라이언트 1개를 `read`로 연결해 조회 경로를 확인합니다.
2. 제어된 워크스페이스에서 `read,write`를 열고 수정 경로를 검증합니다.
3. 동시 작업 구간에 `safe_update_document`와 lease 패턴을 적용합니다.
4. 수명주기 정책 승인 후에만 삭제 권한을 부여합니다.
5. CI에 E2E를 연결해 회귀를 지속적으로 감시합니다.
