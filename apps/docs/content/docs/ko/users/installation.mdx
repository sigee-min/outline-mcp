## 사전 준비

설치 전에 아래 항목을 준비하세요.

- Node.js 20 이상
- npm 또는 pnpm
- Outline API 키

## 옵션 A: npm 설치 (권장)

대부분의 사용자에게 가장 빠른 방법입니다.

```bash
npm i -g @sigee-min/outline-mcp
outline-mcp
```

글로벌 설치 없이 바로 실행하려면 아래를 사용하세요.

```bash
npx -y @sigee-min/outline-mcp
```

## 옵션 B: 소스 실행 (모노레포)

로컬 커스터마이징이나 개발이 필요한 경우 아래 순서를 사용합니다.

```bash
git clone <repo-url>
cd outline-mcp
pnpm install
pnpm --filter @sigee-min/outline-mcp build
pnpm --filter @sigee-min/outline-mcp dev
```

## 필수 환경변수

MCP 서버 실행에는 `OUTLINE_API_KEY`가 필수입니다.

```bash
OUTLINE_API_KEY=your_api_key
OUTLINE_BASE_URL=https://app.getoutline.com
OUTLINE_ALLOWED_ACTIONS=read,write,delete
```

Self-hosted 환경이라면 `OUTLINE_BASE_URL`을 내부 엔드포인트로 바꾸세요.

## 빠른 동작 확인

빌드 후 도구 노출을 확인하려면:

```bash
pnpm --filter @sigee-min/outline-mcp build
pnpm -s dlx @modelcontextprotocol/inspector --cli node apps/mcp/dist/index.js --method tools/list
```
