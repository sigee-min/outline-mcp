## 기본 설정

아래 설정을 기준으로 시작하세요.

```json
{
  "mcpServers": {
    "outline": {
      "command": "npx",
      "args": ["-y", "@sigee-min/outline-mcp"],
      "env": {
        "OUTLINE_API_KEY": "your_api_key",
        "OUTLINE_BASE_URL": "https://app.getoutline.com",
        "OUTLINE_ALLOWED_ACTIONS": "read,write,delete"
      }
    }
  }
}
```

## 안전한 롤아웃 프로파일

### 읽기 전용 프로파일

```json
{
  "OUTLINE_ALLOWED_ACTIONS": "read"
}
```

### 읽기/쓰기 프로파일 (삭제 제외)

```json
{
  "OUTLINE_ALLOWED_ACTIONS": "read,write"
}
```

### Self-hosted 프로파일

```json
{
  "OUTLINE_BASE_URL": "https://outline.your-company.internal"
}
```

## 연결 후 검증 순서

설정 후 아래 순서로 점검하면 문제를 빠르게 찾을 수 있습니다.

1. `list_collections`
2. `search_documents`
3. `read_document`
4. `create_document`, `update_document` (쓰기 허용 이후)
5. lifecycle/batch 도구 (운영 정책 승인 이후)
