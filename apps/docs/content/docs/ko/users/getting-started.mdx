## 이 페이지 대상

- outline-mcp를 처음 도입하는 사용자
- 안전한 초기 롤아웃 전략이 필요한 팀
- 추상 설명보다 실무 기준이 필요한 운영자

## outline-mcp가 제공하는 것

outline-mcp는 Outline 작업을 MCP 도구로 노출해, 에이전트가 명시적 계약에 따라 동작하게 만듭니다.

활용 범위:

- 컬렉션/문서 워크플로
- 템플릿 기반 생성 흐름
- 멤버십/협업 제어
- export 및 감사 추적

## 안전한 첫 롤아웃

1. read-only로 클라이언트 1개 연결
2. 기본 조회(`list_collections`, `read_document`) 검증
3. `read,write` 확장
4. 삭제 정책 합의 후 `delete` 허용

## 최소 요구사항

- Node.js 20+
- pnpm 10+ (모노레포 사용 시)
- Outline API 키
- MCP 클라이언트 1개 (Codex, Claude Desktop, Cursor 등)
