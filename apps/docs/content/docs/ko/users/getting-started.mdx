## 무엇을 해결하나

outline-mcp는 Outline API를 MCP 도구로 노출해, 에이전트가 컬렉션/문서를 명시적 계약에 따라 다룰 수 있게 합니다.

## 핵심 목표

- 런타임 권한을 명확히 유지
- Outline Cloud와 self-hosted를 동일하게 지원
- 멀티 에이전트 편집 충돌 최소화
- 테스트 가능한 운영 워크플로 제공

## 시작 전 준비

- Node.js 20+
- pnpm 10+
- Outline API 키
- MCP 클라이언트(Codex, Claude Desktop, Cursor 등)

## 첫 번째 목표

먼저 read-only로 1개 클라이언트를 연결하세요.

```bash
OUTLINE_ALLOWED_ACTIONS=read
```

검증 후 `read,write` 또는 `read,write,delete`로 확장하는 것을 권장합니다.
