## 핵심 변수

| 변수 | 기본값 | 변경 시점 |
|---|---|---|
| `OUTLINE_API_KEY` | 필수 | 항상 설정 |
| `OUTLINE_BASE_URL` | `https://app.getoutline.com` | self-hosted 사용 시 |
| `OUTLINE_ALLOWED_ACTIONS` | `read,write,delete` | 에이전트 역할 분리 시 |
| `OUTLINE_REQUEST_TIMEOUT_MS` | `15000` | 네트워크 지연/정책 강화 시 |
| `OUTLINE_RETRY_COUNT` | `2` | 업스트림 불안정 구간 대응 시 |
| `OUTLINE_ENABLE_CAPABILITY_PROBE` | `true` | probe 비활성화가 필요한 특수 상황 |

## Lease 변수

| 변수 | 기본값 | 설명 |
|---|---|---|
| `OUTLINE_LEASE_STRATEGY` | `auto` | `auto`, `memory`, `data_attribute` |
| `OUTLINE_LEASE_ATTRIBUTE_ID` | 없음 | `data_attribute`에서 필수 |
| `OUTLINE_LEASE_DEFAULT_TTL_SECONDS` | `600` | 기본 lease TTL |
| `OUTLINE_LEASE_MAX_TTL_SECONDS` | `7200` | lease 최대 TTL |

## 실무 프로파일 예시

### Read-only 어시스턴트

```bash
OUTLINE_ALLOWED_ACTIONS=read
OUTLINE_ENABLE_CAPABILITY_PROBE=true
```

### Writer

```bash
OUTLINE_ALLOWED_ACTIONS=read,write
```

### Operator

```bash
OUTLINE_ALLOWED_ACTIONS=read,write,delete
```

### 다중 프로세스 lease 제어

```bash
OUTLINE_LEASE_STRATEGY=data_attribute
OUTLINE_LEASE_ATTRIBUTE_ID=your_data_attribute_id
```
