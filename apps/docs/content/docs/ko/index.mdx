outline-mcp 문서에 오신 것을 환영합니다.

이 문서는 데모용이 아니라 실제 운영 환경에서 바로 적용할 수 있도록 작성되었습니다.

## 여기서 할 수 있는 것

- MCP 클라이언트를 빠르게 연결
- 권한 경계(`read`, `write`, `delete`)를 안전하게 설정
- 문서/컬렉션 워크플로를 실무 기준으로 운영
- 멀티 에이전트 충돌/lease 흐름 관리
- 배포 전 E2E 검증
