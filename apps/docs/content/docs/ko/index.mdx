outline-mcp 사용자 문서에 오신 것을 환영합니다.

이 가이드는 권한 경계, 멀티 에이전트 안전성, 재현 가능한 테스트를 기준으로 실제 운영 환경에서 서버를 적용할 수 있도록 구성되어 있습니다.

## 권장 순서

1. **시작 가이드**부터 확인
2. **설치**와 **MCP 클라이언트 연결** 완료
3. 운영 정책에 맞게 **권한 모델** 적용
4. 일상 운영은 **도구 레퍼런스** 기준으로 진행
5. 배포 전 **테스트/E2E** 실행

- 사용자 가이드: `/ko/docs/users`
- 영어 문서: 상단 언어 스위처에서 전환
