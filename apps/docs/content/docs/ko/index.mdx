`outline-mcp` 공식 문서에 오신 것을 환영합니다.

이 문서는 데모 중심 문서가 아니라, 실제 팀이 운영 환경에서 빠르고 안전하게 도입할 수 있도록 작성되었습니다.

## 이 프로젝트로 할 수 있는 일

`outline-mcp`를 사용하면 MCP 클라이언트에서 Outline을 구조적으로 다룰 수 있습니다.

대표적으로 다음을 수행할 수 있습니다.

- 읽기 전용 모드에서 안전한 검색/조회 자동화
- 문서 생성, 수정, 이동, 보관, 복원, 배치 처리
- 템플릿, 댓글, 멤버십, export, 감사 이력 운영
- revision 검증과 lease를 이용한 동시 작업 제어
- resource URI를 통한 컬렉션/문서 컨텍스트 직접 읽기

## 문서 범위

이 문서에서는 다음을 다룹니다.

- 설치 및 MCP 클라이언트 연결
- 런타임 설정과 권한 설계
- 기능별 도구 레퍼런스
- 멀티 에이전트 운영 패턴
- 로컬/CI 테스트 및 inspector E2E
- 운영 중 자주 발생하는 문제 해결
